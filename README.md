# 2048 Game

一个经典的 2048 益智游戏的现代网页实现。

## 🎮 游戏简介

2048 是一款流行的数字益智游戏。玩家通过滑动合并相同数字的方块，目标是合成 2048 方块。

## ✨ 功能特性

- **经典玩法** - 标准 4×4 棋盘，合并相同数字方块
- **多种操作方式** - 支持键盘方向键和触摸滑动
- **分数系统** - 实时分数显示，最高分本地存储
- **撤销功能** - 支持撤销上一步操作
- **主题切换** - Soft（柔和）和 Contrast（高对比度）两种主题
- **音效系统** - 可开关的游戏音效
- **流畅动画** - 方块移动和合并动画效果
- **响应式设计** - 适配不同屏幕尺寸

## 🎯 游戏规则

1. 使用方向键（↑↓←→）或滑动屏幕移动所有方块
2. 相同数字的方块碰撞时会合并成一个更大的数字
3. 每次移动后会随机生成一个新方块（2 或 4）
4. 目标是合成 2048 方块
5. 当棋盘填满且无法移动时，游戏结束

## 🚀 快速开始

### 在线游玩

直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/zym9863/2048.git

# 进入目录
cd 2048

# 用浏览器打开
open index.html
```

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 游戏逻辑（原生 JS，无依赖）
- **Web Audio API** - 音效系统
- **LocalStorage** - 数据持久化

## 🎨 主题预览

### Soft 主题（默认）
柔和的新拟态设计风格，适合长时间游戏。

### Contrast 主题
高对比度设计，提供更清晰的视觉体验。

## 📱 操作说明

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 移动方块 | 方向键 ↑↓←→ | 滑动屏幕 |
| 新游戏 | 点击 "New Game" 按钮 | 同左 |
| 撤销 | 点击 "Undo" 按钮 | 同左 |
| 切换主题 | 点击 "Theme" 按钮 | 同左 |
| 开关音效 | 点击 "Sound" 按钮 | 同左 |

## 💾 数据存储

游戏使用浏览器的 LocalStorage 保存以下数据：

- 最高分（`game2048_best_score`）
- 主题偏好（`game2048_theme`）
- 音效设置（`game2048_sound_enabled`）
